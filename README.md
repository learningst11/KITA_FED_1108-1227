## 3. 기존 공식홈페이지 분석 & 리뉴얼 기획의도
기존 다량의 컨텐츠 및 이미지가 과도한 스크롤 생성하여 사용성 및 사용자의 핵심정보 파악을 방해.<br>
↓<br>
주요 기능은 살리되, 사용자 입장에서 핵심 정보가 한눈에 파악 가능하도록 최적화.<br><br>
